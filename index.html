    Welcome to Keentivate.

    I have some bad news. Keen.io's javascript SDK is a wee bit confusing. A wee bit, as in if you want to build any sort of intricate analytics dashboard, you're going to find yourself in a giant mess of spaghetti code. Keentivate solves that. Keentivate is an almost no-code javascript library that turns your HTML markup into Keen charts

    Install

    Install is easy. All you have to do is include the keentivate javascript file, establish your API keys, and you're golden. The code will look something like below.

    <script type="text/javascript" src="/js/keentivate.min.js"></script>
    <script type="text/javascript">
      var keenCharts = new keentivate("50f9e03d38433113c1000000", "1c6b79aedef24363ba84f6790e81c4de", {
        keenClass: "keentivate"
      })
    </script>

    Examples

    Note: All of these examples use the provided football example data.

    Number Chart

     <div class="keentivate"
            keen-event="Plays"
            keen-filter="field_goal_was_good=Y|play_type=FGXP"
            keen-label="Successful Field Goals">
    </div>

    Line Chart

     <div class="keentivate"
       keen-type="line"
       keen-event="Plays"
       keen-filter="field_goal_was_good=Y|play_type=FGXP"
       keen-label="Successful Field Goals"
       keen-end="2013-01-19T10:00Z"
       keen-start="2013-01-18T18:00Z"
       keen-interval="hourly"
       keen-width="1000"
       keen-title="Successful Field Goals"
       keen-x="time"
       keen-y="field goals">
     </div>

    MultiLine Chart

    <div class="keentivate"
  keen-type="multiline"
  keen-event="Plays"
  keen-filter="field_goal_was_good=Y|play_type=FGXP"
  keen-label="Successful Field Goals"
  keen-end="2013-01-19T10:00Z"
  keen-start="2013-01-18T18:00Z"
  keen-interval="hourly"
  keen-width="1000"
  keen-group="game.home_team"
  keen-title="Successful Field Goals by Team"
  keen-x="time"
  keen-y="field goals">
</div>

    Pie Chart

     <div class="keentivate"
       keen-type="pie"
       keen-event="Plays"
       keen-group="play_type"
       keen-title="Types of Plays">
    </div>

    Funnel Chart

     <ul class="keentivate" keen-type="funnel" keen-watch="game.home_team" keen-title="This chart makes no sense, but this is the data I had...">
       <li keen-event="Plays" keen-filter="interception_player?true"></li>
       <li keen-event="Plays" keen-filter="points_scored=3"></li>
     </ul>
